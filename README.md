# Gen-AI-Playground
The purpose of this project is to do a generative AI playground to compare different image models by inserting a text describing the image that we want to create.

This project is done in cooperation with Verda.
## Instructions
Install [Docker Desktop](https://docs.docker.com/get-started/introduction/get-docker-desktop/) on your computer.

Run the following command in folder **gen-ai-playground** to start the application:

```docker compose up```

To run the backend unit tests:
```docker compose build backend-tests```
```docker compose run --rm backend-tests```
